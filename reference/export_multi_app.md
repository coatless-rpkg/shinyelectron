# Export multi-app Shiny suite as Electron application

Export multi-app Shiny suite as Electron application

## Usage

``` r
export_multi_app(
  appdir,
  destdir,
  config,
  runtime_strategy = NULL,
  sign = FALSE,
  platform = NULL,
  arch = NULL,
  icon = NULL,
  overwrite = FALSE,
  build = TRUE,
  run_after = FALSE,
  open_after = FALSE,
  verbose = TRUE
)
```
